# discord-lilabot

This is a simple discord bot that takes in prompts from the chat and responds to them with the help of an ollama backend when certain triggeers are hit.

---

### tech stack:
python. and discord library ig.

### deployment:

if for some reason you do want to deploy this, just
1) clone this repo locally
2) spin up the uvicorn and ollama daemon backend
3) set the client token etc in the env file correctly
4) sit and enjoy!
